Android Studio 快捷键收集

星期六 28 三月 2015
By hnusr

快捷键太多，只大概列出一些，其余的边在工作中边尝试了

代码编辑方面

  1. Alt + 回车 导入包,自动修正，改变属性private,public等
  2. Ctrl + Alt + L 格式化代码
  3. Ctrl + Alt + O 优化导入的类和包
  4. Alt + Insert 生成代码(如get,set方法,构造函数等)
  5. Ctrl + Space 代码提示，在声明一个变量时代码完成特性会给你显示一个建议名。开始键入“ private FileOutputStream ”
  6. Ctrl + Shift + Space 自动补全代码，实例化一个已知类型对象时很有用
  7. Shift + F6 重构-重命名
  8. Ctrl + X 删除行
  9. Ctrl + D 复制行
  10. Ctrl + / 或 Ctrl + Shift + / 注释（// 或者 / /）
  11. Ctrl + J 自动代码，执行一些你记不起来的 Live Template 缩写，比方输入st后再按
  12. Ctrl + O 可以很容易地重载基本类地方法
  13. Ctrl + Shift + J 快捷键把两行合成一行并把不必要的空格去掉以匹配你的代码格式
  14. Ctrl + Alt + V 简化代码中复杂的声明，提取临时变量

代码提示

  1. Ctrl + Alt + Space 可以完成在或不在当前文件里的类名
  2. Ctrl + P 如果光标置于一个方法调用的括号间，方法参数提示
  3. Ctrl + Q 要快速查看编辑器脱字符处使用的类或方法的Java文档
  4. Alt + Q 显示当前类名

窗口功能跳转方面

  1. Ctrl + E 最近打开的文件
  2. Alt + Shift + C 最近更改的代码
  3. Alt + 1 快速打开或隐藏工程面板
  4. Alt + F1 要在任何视图（ Project View 工程视图， Structure View 结构视图或者其它视图）里快速选择当前编辑地部分（类，文件，方法或者字段）
  5. Alt + left/right 切换代码视图
  6. Ctrl + B 快速打开光标处的类或方法
  7. F2 或 Shift + F2 高亮错误或警告快速定位
  8. Ctrl + F12 可以在当前编辑的文件中快速导航
  9. Ctrl + Alt + Down|Up 快捷键则可以在编译器错误信息或者查找操作结果间跳转。
  10. Alt + Up/Down 在方法间快速移动定位
  11. Ctrl + Home/End 光标跳转到第一行或最后一行下
  12. Ctrl + Shift + Backspace 让你调转到代码中所做改变的最后一个地方
  13. F12 把焦点从编辑器移到最近使用的工具窗口
  14. Escape 把焦点从任意工具窗口移到编辑器
  15. Shift + Escape 不仅可以把焦点移到编辑器上而且还可以隐藏当前（或最后活动的）工具窗口。

查找替换引用方面

  1. Ctrl + F 查找文本
  2. Ctrl + R 替换文本
  3. Ctrl + Shift + Alt + N 查找类中的方法或变量
  4. Alt + F7 查找到在整个工程中使用地某一个类、方法或者变量的位置

其它

  1. Ctrl + Shift + T 建立本类的测试文件
  2. Ctrl + Alt + T 弹出 Code | Surround with… 窗口
  3. Ctrl + Alt + Shift + T 在变量或方法上弹出重构窗口
  4. Alt + F3 在编辑器中实现快速查查找功能
  5. Ctrl + N, Ctrl + Shift + N 快速找类或文件
  6. Ctrl + Shift + V 剪贴板内容选择插入到文本
  7. Alt + F8 debug模式下计算变量值
  8. F11, Shift + F11 给当前行打标签，如果是数字标签，则Ctrl + 数字直接跳转，如果是普通标签，Shift + F11弹出窗口查看
  9. Ctrl + Shift + I 窗口中显示当前方法的详细内容，不需要像Ctrl + B 那样去跳转到新的类，适合快速查看之用
  10. Ctrl + Alt + H 显示一个方法被调用的层次结构
